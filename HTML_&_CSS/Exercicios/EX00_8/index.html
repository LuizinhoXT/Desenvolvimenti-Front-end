Lista ordenada

Usamos a tag (Ordered List) <ol>para representar uma lista ordenada; Dentro da tag, temos o <li> que são os itens da lista;

  1. item 1
  2. item 2
  3. item 3
    1. sub-item 1 / 3.1
    2. sub-item 2 / 3.2
    3. sub-item 3 / 3.3

Lista Não Oordenada

Em lista não ordenadas, não temos a logica de nuerar, em vez disso, iremos usar marcadores (bolinhas)

  • item 1
  • item 2
  • item 3
    • sub-item 1 / 3.1
    • sub-item 2 / 3.2
    • sub-item 3 / 3.3

Plataformas de streaming

É possível utilizar a tag <a<, que possui um atributo chamado href. Esse atributo é usado para colocar um hyperlink em cima de um texto.

Dentro do href, podemos usar o simbolo # para referenciar ao id de alguma tag. Essa referência precisa ter o mesmo id da tag.

  • Netflix
