Principais Formatações


Forma ou sentido?

O HTML5 é mais indicado para definir o significado das tags dentro da estrutura, deixando de ser um mero ajuste visual.

Atualmente, usamos o HTML5 para cuidar da semântica, ou seja, significado dos elementos de um site, do que se tratam, e qual é a estrutura do conteúdo; enquanto que o CSS (Folhas de Estilo em Cascata) é usado para tratar da forma visual do site

Negrito / Destaque

Esse texto foi escrito usando a tag "Bold" ⇒ <b>, e figando em negrito (forma)

nessa frase temos um Termo Destaque escrito utilizando a tag <strong> (Semântica)

Itálico / Ênfase

nessa frase, temos um termo em itálico, utilizando a tag <i> (forma)

nessa frase, temos um temos em enfase, utilizando a tag <em> (Semântica)

Texto Marcado

Em textos grandes, é legal marcar um texto utilizando a tag <mark>, (Semântica)

Nessa marcação na cor blueviolet, foi usado um atributo dentro da tag <mark> chamado style, sendo assim, dentro do Style, conseguimos definir uma cor especifica. Isso ocorre pois o Style (CSS) é usado para definir a forma visual do conteúdo

Texto Inserido

Podemos marcar um texto com a tag <ins> como inserido para que signifique que ele fora incluído depois

Texto Deletado

utilizamos a tag <del> que dá o sentido de que o texto deve ser lido, porém não deve ser considerado

Texto Sobreescrito

Podemos usar a tag <sup> x20 ou 216; para marcar notações matemáticas 12

Texto Subscrito

Já a tag <sub> é usado para marcar texto sobreescrito, como fórmulas químicas ou logarítmicas: log10256 ,H2O
