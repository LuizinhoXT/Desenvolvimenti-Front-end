Adicionando Emoji no site: 🛹

Uma maneira é ir até o site: emojipedia.org

lá temos uma enciclopédia de emojis que podemos pesquisar por palavra-chave relacionada e copiar e colar no index. O problema é que não funciona em todos os navegadores.

Uma solução é adicionar o codepoint do emoji. Quando adiciono o seguinte emoji: 🔥, de fogo, é isso que faço e utilizo o apenas os números do codepoint: U+1F525

curiosidade: Parênteses angular é o nome dessas paradinhas → "< >"
