Hiper Text Markup Language é a tecnologia usada para os navegadores para disponibilizar textos que não possuam uma leitura linear dentro de uma página da web. Essa leitura caracteriza o HTML pois Utilizamos links no meio da comunicação textual e isso torna a leitura em algo não linear.

Os navegadores usam o protocolo HTTP para manipular o conteúdo em HTML e fazer o intercâmbio das informações.

A estrutura básica do HTML é se baseia em tags. Essas tags são utilizadas para marcar partes específica como títulos, cabeçalho, rodapé e etc.. Com uma boa estrutura logicamente aplicada, o código HTML se torna funcional em diversos tipos de navegadores.

Principais tags

<html> <html/>

Essa tag é utilizada para marcar o inicio e fim do documento, levando o leitor do documento a entender que se trata de um documento em html

<head> <head/>

Tem a função de delimitar o cabeçalho da página, ou seja, nessa seção poderão conter, por exemplo, o título e o autor da página em questão. Neste contexto, você poderá usar as tags <tittle> e “meta tags”, respectivamente.

<body> <body/>

A utilização desta tag serve para delimitar o conteúdo da página propriamente dito. Entre a marcação de início e a marcação de finalização, encontram-se, por exemplo, o texto, os links e as tabelas da página.

<blockquote> <blockquote/>

O Elemento HTML blockquote (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.

Headings

São tags em HTML iniciadas com H e váo de 1 até 6. Eles representam os níveis de título em HTML

temos:
  1. titulo principal como a primeira coisa a ser vista em um texto
  2. subtítulo um segundo nível, falando do assunto de cada conteúdo
  3. nível 3 que está sempre depois no nível 2 e assim por diante

*Não é recomendado pular os níveis de título. Deve-se sempre respeitar a hierarquia, pois isso irá tornar o texto mais legível.

Texto Sobreescrito

Podemos usar a tag <sup> x20 ou 216; para marcar notações matemáticas 12

Texto Subscrito

Já a tag <sub> é usado para marcar texto sobreescrito, como fórmulas químicas ou logarítmicas: log10256 ,H2O
