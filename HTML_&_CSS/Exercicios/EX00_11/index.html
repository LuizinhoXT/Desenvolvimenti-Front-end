coisas importantes sobre o Formulário:

  • A tag <form> possui atributos:
    • nome: nome do formulário, lembrando a orientação a objeto
    • action: local para onde os dados do formulário será enviado, snedo uma URL de um servidor
  • Os campos são como variáveis abertas onde o usuário pode inserir dados. isso significa que a tag <input> possui atributos e tipagem como:
    • nome
    • identificador
    • tipo
  • recuperação de dados ocorre localizando os dados através de chaves como nome, identificador e valor armazenado.
